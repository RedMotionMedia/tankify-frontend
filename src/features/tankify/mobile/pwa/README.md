# Mobile PWA (Step 2)

Dieser Ordner ist der Anker fuer die spaetere Umstellung der Mobile-UI auf eine Progressive Web App.

Geplanter Inhalt in Schritt 2 (ohne Anspruch auf Vollstaendigkeit):
- App Manifest (`src/app/manifest.ts`) + Icons in `public/`
- Service Worker (z.B. `public/sw.js`) fuer Offline-Caching/Runtime-Cache
- Offline-Fallback (Route/Seite) und Cache-Strategien fuer Karten/Stationen
- Install-UX (optional) und Push/Background Sync (optional)
